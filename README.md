# AI Learning Assistant

## Overview
AI-powered learning assistant that detects learner emotions and provides personalized guidance.

## Technologies
- Python
- Streamlit
- TensorFlow/Keras
- BiLSTM
- BERT
- Gemini AI
- Plotly
- NLTK

## Features
- Emotion Detection
- Model Comparison (BiLSTM vs BERT)
- AI Learning Guidance
- CSV Logging
- Analytics Dashboard

## Run

pip install -r requirements.txt

streamlit run app.py
